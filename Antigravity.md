# Antigravity AI Assistant Context & Workflow

## Objetivo Principal
Este documento serve como a base de contexto e diretrizes para o desenvolvimento contínuo do projeto **Muscleshop.ai**. O foco principal é a integração fluida entre automação de processos, gerenciamento de banco de dados e melhorias no CRM.

## Tecnologias e Ferramentas Core
- **n8n**: Criação, edição e gerenciamento de fluxos de trabalho (workflows) automatizados via MCP.
- **Supabase**: Gerenciamento do banco de dados relacional, autenticação e funções edge via MCP.
- **CRM Muscleshop.ai**: Desenvolvimento e aprimoramento contínuo das funcionalidades e da interface do usuário (UI/UX).

## Repositórios de Skills (Em breve)
Para maximizar a eficiência, integraremos e utilizaremos as seguintes bibliotecas de skills:

2. **[n8n Agent Skills](https://github.com/czlonkowski/n8n-skills)**: Para criação inteligente e manutenção de workflows no n8n.

4. **[Obra Superpowers](https://github.com/obra/superpowers)**: Ferramentas adicionais para expandir as capacidades de desenvolvimento e análise.

## Fluxo de Trabalho (Workflow)
1. **Planejamento**: O usuário descreve a melhoria desejada (ex: novo fluxo no n8n que salva dados no Supabase e reflete no CRM).
2. **Análise**: O assistente (Antigravity) analisa os requisitos, verifica a estrutura atual e planeja as ações.
3. **Execução Integrada**:
   - **Banco de Dados**: Modificações no schema, tabelas ou functions do Supabase.
   - **Automação**: Construção ou atualização de workflows no n8n, garantindo o fluxo correto de dados.
   - **Frontend/Backend (CRM)**: Modificação do código-fonte para implementar a interface do usuário e a lógica de negócio, focando sempre em um design dinâmico e premium.
4. **Revisão e Testes**: Validação conjunta das funcionalidades.

## Diretrizes de Design (UX/UI) para o CRM
- **Aparência Premium**: O design deve ser moderno e atraente.
- **Cores e Tipografia**: Uso de paletas harmoniosas e fontes limpas (ex: Inter, Outfit).
- **Interatividade**: Efeitos visuais modernos como glassmorphism, dark mode inteligente e micro-animações.
